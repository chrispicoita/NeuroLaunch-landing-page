Guarded rollouts with automatic rollback

          
Define thresholds across success, latency, cost, and toxicity. If a new prompt violates SLOs during rollout, NeuroLaunch automatically rolls back and alerts your team.


          

            
              See SDK usage
              
                
              
            
            

              

                
                
Python

                
import neurolaunch as nl
run = nl.eval(dataset='support_tickets_golden', metrics=['task_success','latency_ms'])
if run.passes({'task_success':0.85}):
  nl.rollout(prompt_version='v19', to='20%')
else:
  nl.rollback()

              

              

                
                
JavaScript

                
import nl from 'neurolaunch';
const run = await nl.eval({ dataset: 'support_tickets_golden', metrics:['task_success'] });
if (run.passes({ task_success:0.85 })) {
  await nl.rollout({ promptVersion:'v19', to:'20%' });
} else {
  await nl.rollback();
}
